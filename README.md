Portfólio pessoal desenvolvido com Next.js, React, TypeScript e Tailwind CSS, projetado para apresentar minha trajetória, habilidades técnicas, experiência profissional e projetos. O site conta com interface responsiva, animações fluidas utilizando Framer Motion, tema moderno e formulário de contato integrado via EmailJS, oferecendo uma experiência rápida, elegante e otimizada para recrutadores e clientes.
